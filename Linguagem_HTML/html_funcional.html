Comandos de Funcionalidade

Comandos de funcionalidade são comandos que normalmente são colocados antes e dentro do head. Eles têm funções que não interferem diretamente no conteúdo e na estruturação da página, mas possuem propriedades a serem lidas pelo navegador.

Abaixo, irei citar alguns desses comandos e algumas de suas variações:

link
A tag link cria uma ligação entre o documento citado dentro da tag com o presente documento. Essa ligação faz com que os dados do documento citado possam se manifestar e alterar as propriedades do presente documento.

Um exemplo prático para esta tag poderia ser:

Código:

<link rel="stylesheet" href="style.css">
<link rel="shortcut icon" href="./assets/img/Favicon2.png" type="image/x-icon">

Resultado:

Não aparece resultado na tela.

Esta tag possui alguns atributos que podem ser utilizados como:

O atributo href indica o endereço do documento que será ligado.

O atributo rel indica qual a relação que o documento citado terá com o presente documento.

O atributo type especifica o tipo de mídia do documento.

meta
A tag meta define uma série de dados sobre o documento.

Um exemplo prático para esta tag poderia ser:

Código:

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="author" content="Eu">

Resultado:

Não aparece resultado na tela.

Esta tag possui alguns atributos que podem ser utilizados como:

O atributo charset define os tipos de caracteres que o documento poderá renderizar.

O atributo content define os valores que serão associados aos atributos name e http-equiv.

O atributo http-equiv utilizei pouco então não sei direito o que faz.

O atributo name escolhe um tipo de nome para os meta dados.

style
A tag style cria uma seção de código CSS dentro do próprio documento.

Um exemplo prático para esta tag poderia ser:

Código:

<style>
    mark {
        background-color: lime;
    }
</style>

Resultado:

Eu não vou colocar o resultado visual pois pode vir a comprometer outras partes da página, mas tudo que estivesse dentro da tag mark ficaria marcado com verde limão.

Esta tag possui alguns atributos que podem ser utilizados como:

O atributo media pode ser usado para definir em que tipos de mídia ou tamanhos de tela uma determinada estilização será visível.

title
A tag title é usada para dar nome ao título do documento que será visível na página de busca do navegador e na aba do navegador.

Um exemplo prático para esta tag poderia ser:

Código:

<title>Comandos de Funcionalidade</title>

Resultado:

Está visível na aba da página no navegador.
