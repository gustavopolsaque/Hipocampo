Comandos Estruturais

Comandos estruturais são comandos que criam separações de conteúdo nas páginas.

Abaixo, irei citar alguns desses comandos e em que situações são usados:

article
A tag article serve para englobar um artigo independente do conteúdo da página.

Um exemplo prático para esta tag poderia ser:

Código:

<article>
    <p>O projeto HIPOCAMPO teve início no mês de outubro de 2023.</p>
</article>

Resultado:

O projeto HIPOCAMPO teve início no mês de outubro de 2023.

aside
A tag aside engloba os conteúdos à parte da página, sendo como um conteúdo secundário para complemento.

Um exemplo prático para esta tag poderia ser:

Código:

<aside>
    <p>O <code>aside</code> faz parte das tags que são possíveis de se utilizar com HTML.</p>
</aside>

Resultado:

O aside faz parte das tags que são possíveis de se utilizar com HTML.

body
A tag body é o corpo do documento e engloba todo conteúdo visível do documento.

Um exemplo prático para esta tag poderia ser:

Código:

<body>
    <p>Estou dentro do body</p>
</body>

Resultado:

Estou dentro do body

col
A tag col é usada dentro de uma tabela junto do colgroup para criar um tipo de estilização por coluna, a função deste é dizer quais estilizações a coluna terá.

Um exemplo prático para esta tag poderia ser:

Código:

<table style="border: 1px solid black;">
    <colgroup>
        <col span="2" style="text-align: center;">
    </colgroup>
    <tr>
        <td style="border: 1px solid black;">Celula 1</td>
        <td style="border: 1px solid black;">Celula 2</td>
    </tr>
</table>

Resultado:

Celula 1 Celula 2

Esta tag possui alguns atributos que podem ser utilizados como:

O atributo span permite com que a estilização seja aplicada a mais de uma coluna.

colgroup
A tag colgroup é usada dentro de uma tabela junto do colgroup para criar um tipo de estilização por coluna, a função dela é reunir as configurações das colunas de uma tabela.

Um exemplo prático para esta tag poderia ser:

Código:

<table style="border: 1px solid black;">
    <colgroup>
        <col span="2" style="text-align: center;">
    </colgroup>
    <tr>
        <td style="border: 1px solid black;">Celula 1</td>
        <td style="border: 1px solid black;">Celula 2</td>
    </tr>
</table>

Resultado:

Celula 1 Celula 2

Esta tag possui alguns atributos que podem ser utilizados como:

O atributo span permite com que a estilização seja aplicada a mais de uma coluna.

div
A tag div é usada para criar divisões no documento englobando outras tags para facilitar estilizações.

Um exemplo prático para esta tag poderia ser:

Código:

<div>
    <p>Este texto esta dentro de uma divisão.</p>
</div>

Resultado:

Este texto esta dentro de uma divisão.

dl
A tag dl é usada juntamente com as tag dt e dt para criar uma lista de definições, sua função neste tipo de lista agrupar toda lista de definição.

Um exemplo prático para esta tag poderia ser:

Código:

<dl>
    <dt>Água</dt>
    <dd>Liquido transparente composto de H<sub>2</sub>O</dd>
</dl>

Resultado:

Água
Liquido transparente composto de H2O
fieldset
A tag fieldset agrupa elementos dentro de um formulário.

Um exemplo prático para esta tag poderia ser:

Código:

<fieldset name="texto">
    <legend>Formulário</legend>
</fieldset>

Resultado:

Formulário

Esta tag possui alguns atributos que podem ser utilizados como:

O atributo disabled desabilita os campos do formulário.

O atributo form faz uma conexão entre o formulário e um conjunto de campos.

O atributo name adiciona um nome ao formulário.

footer
A tag blockquote é o ródape do documento.

Um exemplo prático para esta tag poderia ser:

Código:

<footer>
    <p>Ródape do documento.</p>
</footer>

Resultado:

Ródape do documento.

form
A tag form é usado para emglobar um formulário.

Um exemplo prático para esta tag poderia ser:

Código:

<form action="cadastro.php" method="get">
    <fieldset>
        <legend>Dados pessoais</legend>
        <p>
            <label for="inome">Nome:</label>
            <input type="text" name="nome" id="inome">
        </p>
        <p>
            <label for="iemail">E-mail: </label>
            <input type="email" name="email" id="iemail" autocomplete="email" required>
        </p>
        <p>
            <label for="itel">Telefone: </label>
            <input type="tel" name="tel" id="itel" autocomplete="tel" placeholder="(xx)xxxx-xxxx" pattern="^\(\d{2}\)\d{4,5}-\d{4}$" required>
        </p>
    </fieldset>
    <p>
        <input type="submit" value="Enviar">
        <input type="reset" value="Limpar">
    </p>
</form>

Resultado:

Dados pessoais

Esta tag possui alguns atributos que podem ser utilizados como:

O atributo accept-charset Nunca usei.

O atributo action indica para onde sera enviado os dados do formulário.

O atributo autocomplete é booleano e serve para o navegador completar automaticamente com respostas anteriores da página ou usadas frequentemente.

O atributo enctype Nunca usei.

O atributo method define como os dados serão enviados pela URL, se será decodificado (get) ou codificado (post).

O atributo name define o nome do formulário.

O atributo novalidate Nunca usei.

O atributo rel Nunca usei.

O atributo target define como a resposta será exibida na tela.

head
A tag head agrupa os dados que não aparecem no resultado do documento.

Um exemplo prático para esta tag poderia ser:

Código:

                <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Título</title>
                </head>

Resultado:

Não aparece resultado na tela.

header
A tag header é o cabeçalho do documento.

Um exemplo prático para esta tag poderia ser:

Código:

                <header>
                    <p>Cabeçalho</p>
                </header>

Resultado:

Cabeçalho

html
A tag html agrupa QUASE todo código HTML.

Um exemplo prático para esta tag poderia ser:

Código:

                <html>
                    <head>
                        <meta charset="UTF-8">
                    </head>
                    <body>
                        <p>Documento em HTML</p>
                    </body>
                </html>

Resultado:

Documento em HTML

Esta tag possui alguns atributos que podem ser utilizados, como:

O atributo lang indica ao navegador em qual língua a página deve ser lida.

iframe
A tag iframe cria uma janela dentro do documento para outra página.

Um exemplo prático para esta tag poderia ser:

Código:

                <iframe src="../index.html" frameborder="0" scrolling="no"><a href="../index.html" target="_blank">Página inicial do Hipocampo</a></iframe>

Resultado:

Esta tag possui alguns atributos que podem ser utilizados, como:

O atributo allow cria restrições a serem impostas no iframe.

O atributo allowfullscreen define se o iframe pode ficar em tela cheia ou não.

O atributo height altera a altura do vídeo que será reproduzido.

O atributo name define o nome do iframe.

O atributo referrerpolicy é usado para que o iframe possa ou não enviar dados ao ser buscado.

O atributo sandbox impõe restrições de captura de informação ou funções.

O atributo src indica o endereço URL do frame que será buscado.

O atributo srcdoc usa código de HTML para gerar um código dentro do frame.

O atributo width altera a largura do vídeo que será reproduzido.

img
A tag img adiciona uma imagem ao documento.

Um exemplo prático para esta tag poderia ser:

Código:

                <img src="imagem.png" alt="Imagem">

Resultado:

Aparece na tela a primeira imagem que estiver no src.

Esta tag possui alguns atributos que podem ser utilizados, como:

O atributo alt cria um texto alternativo para a imagem para caso ela não apareça ou para leitores de tela.

O atributo height altera a altura do vídeo que será reproduzido.

O atributo src indica o endereço URL do vídeo que será buscado.

O atributo width altera a largura do vídeo que será reproduzido.

main
A tag main é o conteúdo principal do documento.

Um exemplo prático para esta tag poderia ser:

Código:

                <main>
                    <p>Corpo principal</p>
                </main>

Resultado:

Corpo principal

nav
A tag nav engloba um grupo de links para navegação.

Um exemplo prático para esta tag poderia ser:

Código:

                <nav>
                    <a href="#">Mesma página 1</a>
                    <a href="#">Mesma página 2</a>
                    <a href="#">Mesma página 3</a>
                </nav>

Resultado:

Mesma página 1 Mesma página 2 Mesma página 3

Esta tag possui alguns atributos que podem ser utilizados, como:

O atributo cite permite adicionar uma URL para especificar a fonte do trecho citado dentro da tag blockquote.

optgroup
A tag optgroup agrupa opções de um select.

Um exemplo prático para esta tag poderia ser:

Código:

                <form action="cadastro.php" method="post" autocomplete="on">
                    <p>
                        <label for="iest">Estado: </label>
                        <select name="estado" id="iest">
                            <option value="">--- Escolha ---</option>
                            <optgroup label="Região Sudeste">
                                <option value="MG">Minas Gerais</option>
                                <option value="RJ">Rio de Janeiro</option>
                                <option value="SP">São Paulo</option>
                            </optgroup>
                            <optgroup label="Região Nordeste">
                                <option value="MA">Maranhão</option>
                                <option value="PE">Pernambuco</option>
                                <option value="RN">Rio Grande do Norte</option>
                            </optgroup>
                        </select>
                    </p>
                </form>

Resultado:

Esta tag possui alguns atributos que podem ser utilizados, como:

O atributo cite permite adicionar uma URL para especificar a fonte do trecho citado dentro da tag blockquote.

ol
A tag ol é usada juntamente com a tag li para criar uma lista ordenada.

Um exemplo prático para esta tag poderia ser:

Código:

                <ol reversed start="5" type="I">
                    <li>Item 1</li>
                    <li>Item 2</li>
                </ol>

Resultado:

  1. Item 1
  2. Item 2

Esta tag possui alguns atributos que podem ser utilizados, como:

O atributo reversed faz com que a ordem da lista seja ao contrário.

O atributo start indica por qual número a lista deve começar.

O atributo type define que tipo de marcador será utilizado na lista. Os tipos são o numérico (1), alfabeto maiúsculo e minúsculo (A e a) e números romanos maiúsculo e minúsculo (I e i).

picture
A tag picture cria um agrupamento de imagens podendo colocar condições para aparecer diferentes imagens.

Um exemplo prático para esta tag poderia ser:

Código:

<picture>
    <source media="(max-width: 750px)" srcset="imagens/foto-p.png" type="image/png">
    <source media="(max-width: 1050px)" srcset="imagens/foto-m.png" type="image/png">
    <img src="imagens/foto-g.png" alt="Imagem flexivel">
</picture>

Resultado:

Aparece na tela a primeira imagem que estiver no srcset do source e cumprir os requisitos da media.

section
A tag section cria uma seção no documento.

Um exemplo prático para esta tag poderia ser:

Código:

<section>
    <p>Este texto esta dentro de uma seção.</p>
</section>

Resultado:

Este texto esta dentro de uma seção.

table

Um exemplo prático para esta tag poderia ser:

Código:

<table style="border: 1px solid black;">

        <th colspan="2" id="ex" scope="colgroup" style="border: 1px solid black;">cabeçario</th>

            <td colspan="2" style="border: 1px solid black;">A</td>
        </tr>

        </tr>
        <tr>
            <td headers="ex" style="border: 1px solid black;">D</td>
        </tr>
    </tbody>
</table>

A
B C
D

video
A tag video cria um agrupamento de vídeos podendo colocar condições para aparecer diferentes vídeos.

Um exemplo prático para esta tag poderia ser:

Código:

                        <video width="600" poster="imagens.png" controls autoplay loop>
                            <source src="video.mp4" type="video/mp4">
                            <source src="video.m4v" type="video/mp4">
                            <source src="video.webm" type="video/webm">
                            <source src="video.ogv" type="video/ogg">
                        </video>

Resultado:

Aparece na tela o primeiro vídeo que estiver no srcset do source e for o primeiro a ser compatível com o navegador.

Esta tag possui alguns atributos que podem ser utilizados como:

O atributo autoplay faz com que o vídeo seja iniciado automaticamente, mas, em alguns navegadores não funciona a menos que esteja com o atributo muted junto.

O atributo controls faz com que apareça os botões de controle do vídeo.

O atributo height altera a altura do vídeo que será reproduzido.

O atributo loop faz o vídeo se reiniciar assim que acabar.

O atributo muted faz o vídeo ficar sem som até que seja alterado o volume.

O atributo poster altera a tela inicial do vídeo até que seja iniciado.

O atributo preload faz o vídeo carregar apenas quando o usuário interagir com ele.

O atributo src indica o endereço URL do vídeo que será buscado.

O atributo width altera a largura do vídeo que será reproduzido.
