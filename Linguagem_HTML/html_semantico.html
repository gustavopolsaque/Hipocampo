Comandos Semânticos

Comandos semântico são comandos que passam o sentido sintaxico do texto para os navegadores interpretarem e entenderem melhor o conteúdo da página.

Abaixo irei citar alguns desses comandos e em que situações são usados:

abbr
A tag abbr é usada para fazer a definição de uma abreviação.

Um exemplo prático para esta tag poderia ser:

Código:

<abbr title="Brasil">BR</abbr>

Resultado:

BR

Esta tag possui alguns atributos que podem ser utilizados como:

O atributo title é usado para definir qual a abreviação dentro da tag abbr.

address
A tag address é usada para informações de contato seja endereço físico, digital ou telefônico.

Um exemplo prático para esta tag poderia ser:

Código:

<address>Rua das Avenidas 112<br>exemplo@exemplo.exemplo<br>+00 (00) 90000-0000</address>

Resultado:

Rua das Avenidas 112
exemplo@exemplo.exemplo
+00 (00) 90000-0000
audio
A tag audio cria um audio ou um agrupamento de audios usando junto ao source podendo colocar condições para aparecer diferentes audios.

Um exemplo prático para esta tag poderia ser:

Código:

<audio preload="metadata" controls loop>
    <source src="../ProjetosHTMLCSS/CursoemVideo/exercicios/ex011/midia/guanacast-33.mp3" type="audio/mpeg">
    <source src="../ProjetosHTMLCSS/CursoemVideo/exercicios/ex011/midia/guanacast-33.mp3" type="audio/ogg">
    <source src="../ProjetosHTMLCSS/CursoemVideo/exercicios/ex011/midia/guanacast-33.mp3" type="audio/wav">
    <p>Infelizmente seu navegador não consegue reproduzir áudio. <a href="../ProjetosHTMLCSS/CursoemVideo/exercicios/ex011/midia/guanacast-33.mp3">Clique aqui para baixar o arquivo MP3</a>

</audio>

Resultado:

Esta tag possui alguns atributos que podem ser utilizados como:

O atributo autoplay faz com que o audio seja iniciado automaticamente, mas, em alguns navegadores não funciona a menos que esteja com o atributo muted junto.

O atributo controls faz aparecer os controles do audio na tela.

O atributo loop reproduz o audio assim que ele acabar.

O atributo muted faz o audio ficar sem som até que seja alterado o volume.

O atributo preload indica se o audio sera carregado apenas se interagirem, se carrega apenas o arquivo ou assim que a página carregar.

O atributo src indica o endereço URL do audio que será buscado.

blockquote
A tag blockquote é usada fazer citações extensas de outra fonte.

Um exemplo prático para esta tag poderia ser:

Código:

<blockquote cite="endereco_eletronico">Lorem ipsum dolor sit amet consectetur adipisicing elit. Nam quos, delectus aliquid maiores dignissimos eligendi illo laboriosam commodi reprehenderit tempore est mollitia eum error iure numquam molestiae, adipisci officiis quam.</blockquote><address>escrito por Lorem ipsum</address>

Resultado:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Nam quos, delectus aliquid maiores dignissimos eligendi illo laboriosam commodi reprehenderit tempore est mollitia eum error iure numquam molestiae, adipisci officiis quam.
escrito por Lorem ipsum

Esta tag possui alguns atributos que podem ser utilizados como:

O atributo cite deixa adicionar uma URL para expecifica de a fonte do trecho citado dentro da tag blockquote.

br
A tag br é usado para fazer uma quebra de linha dentro de outras tags.

Um exemplo prático para esta tag poderia ser:

Código:

<p>Todo esse texto <br>esta no mesmo parágrafo </p>

Resultado:

Todo esse texto
esta no mesmo parágrafo

caption
A tag caption cria uma legenda/titulo fora da tabela na parte superior.

Um exemplo prático para esta tag poderia ser:

Código:

<table style="border: 1px solid black;">
    <caption>Titulo</caption>
    <tr>
        <td>44</td>
        <td>55</td>
    </tr>
</table>

Resultado:

Titulo
44 55
code
A tag code é usada para citar códigos de conputação

Um exemplo prático para esta tag poderia ser:

Código:

<p>No <abbr title="HyperText Markup Language">HTML</abbr> a tag <code>a</code> serve para criar link</p>

Resultado:

No HTML a tag a serve para criar link

dd
A tag dd é usada juntamente com as tag dl e dt para criar uma lista de definições, sua função neste tipo de lista é dar definição ao termo da lista de definição.

Um exemplo prático para esta tag poderia ser:

Código:

<dl>
    <dt>Água</dt>
    <dd>Liquido transparente composto de H<sub>2</sub>O</dd>
</dl>

Resultado:

Água
Liquido transparente composto de H2O
dt
A tag dt é usada juntamente com as tag dl e dd para criar uma lista de definições, sua função neste tipo de lista ser preenchido com o termo a ser descrito na lista de definição.

Um exemplo prático para esta tag poderia ser:

Código:

<dl>
    <dt>Água</dt>
    <dd>Liquido transparente composto de H<sub>2</sub>O</dd>
</dl>

Resultado:

Água
Liquido transparente composto de H2O
del
A tag del é usada para dizer que um texto foi excluído na página após a primeira publicação.

Um exemplo prático para esta tag poderia ser:

Código:

<p>Fui no mercado e esqueci <del cite="URL.algumacoisa" datetime="2023-11-04T16:58:36Z">a carteira</del> <ins cite="URL.algumacoisa" datetime="2023-11-04T16:59:13Z">o cartão</ins></p>

Resultado:

Fui no mercado e esqueci a carteira o cartão

Esta tag possui alguns atributos que podem ser utilizados como:

O atributo cite deixa adicionar uma URL para expecifica do porque o texto foi excluido.

O atributo datetime permite colocar a data, hora e até citar o fuso horário em que foi excluido o texto

em
A tag em é usada para enfatizar uma frase ou palavra de um texto

Um exemplo prático para esta tag poderia ser:

Código:

<p>Eu <em>pretendo</em> ficar atualizando esse projeto</p>

Resultado:

Eu pretendo ficar atualizando esse projeto

h1 a 6
As tags h1 a h6 são usadas como titulos no documento em grau de hierarquia.

Um exemplo prático para esta tag poderia ser:

Código:

<h6>Este é um h6 e o título da página é um h1</h6>

Resultado:

Este é um h6 e o título da página é um h1
ins
A tag ins é usada para dizer que um texto foi inserido na página após a primeira publicação.

Um exemplo prático para esta tag poderia ser:

Código:

<p>Fui no mercado e esqueci <del cite="URL.algumacoisa" datetime="2023-11-04T16:58:36Z">a carteira</del> <ins cite="URL.algumacoisa" datetime="2023-11-04T16:59:13Z">o cartão</ins></p>

Resultado:

Fui no mercado e esqueci a carteira o cartão

Esta tag possui alguns atributos que podem ser utilizados como:

O atributo cite deixa adicionar uma URL para expecifica do porque o texto foi excluido.

O atributo datetime permite colocar a data, hora e até citar o fuso horário em que foi excluido o texto

legend
A tag legend cria uma legenda/titulo para ser usada em um fieldset que normalmente é usado para formulários.

Um exemplo prático para esta tag poderia ser:

Código:

<fieldset>
    <legend>Formulário vazio</legend>
</fieldset>

Resultado:

Formulário vazio
li
A tag li é usada juntamente com as tags ol ou ul para criar itens nas listas.

Um exemplo prático para esta tag poderia ser:

Código:

<ol>
    <li>Item 1</li>
    <li>Item 2</li>
</ol>

Resultado:

  1. Item 1
  2. Item 2
mark
A tag mark é usada para destacar um termo ou frase.

Um exemplo prático para esta tag poderia ser:

Código:

<p>Ele serve como <mark>marca texto</mark></p>

Resultado:

Ele serve como marca texto

p
A tag p é usada para colocar paragrafos nos documentos.

Um exemplo prático para esta tag poderia ser:

Código:

<p>O céu esta sem nuvens hoje</p>

Resultado:

O céu esta sem nuvens hoje

pre
A tag pre cria um espaço pré-formatado que exibe o texto da forma como está escrito no código.

Um exemplo prático para esta tag poderia ser:

Código:

<p><pre>Tudo que        esta escrito aqui 
    pertence ao mesmo   parágrafo</pre></p>

Resultado:

Tudo que        está escrito aqui 
                        pertence ao mesmo   parágrafo

q
A tag q é usada fazer citações curtas de outra fonte.

Um exemplo prático para esta tag poderia ser:

Código:

<q cite="endereco_eletronico">Lorem ipsum dolor sit amet consectetur adipisicing elit.</q><address>escrito por Lorem ipsum</address>

Resultado:

Lorem ipsum dolor sit amet consectetur adipisicing elit.
escrito por Lorem ipsum

Esta tag possui alguns atributos que podem ser utilizados como:

O atributo cite deixa adicionar uma URL para expecificar a fonte do trecho citado dentro da tag q.

source
A tag source é usado para buscar diferentes formatos de audio, imagem e video retornando o primeiro que for compativel com o navegador.

Um exemplo prático para esta tag poderia ser:

Código:

<source media="(max-width: 750px)" sizes="(min-width: 600px) 500px" srcset="imagem_a_ser_buscada.algo" type="image/algo">

Resultado:

Aparece na tela a imagem que estiver no srcset.

Esta tag possui alguns atributos que podem ser utilizados como:

O atributo media permite que o retorno do source seja diferente dependendo do valor maximo ou minimo da tela.

O atributo sizes permite definir o tamanho da imagem de retorno do source.

O atributo src é onde fica a URL do retorno do source apenas se for usada nas tags audio ou video.

O atributo srcset é onde fica a URL do retorno do source apenas se for usada na tag picture.

O atributo type indica qual sera o formato do arquivo presente no source.

span
A tag span cria uma divisão no meio de um texto para na maioria das vzes colocar uma estilização.

Um exemplo prático para esta tag poderia ser:

Código:

<p>Estou ficando <span style="color: green;">sem ideias</span> para exemplo.</p>

Resultado:

Estou ficando sem ideias para exemplo.

strong
A tag strong é usada para indica que o trecho possui importância.

Um exemplo prático para esta tag poderia ser:

Código:

<p>Eu <strong>preciso</strong> caminhar amanhâ.</p>

Resultado:

Eu preciso caminhar amanhâ.

sub
texto subscrito
A tag sub é usada para fazer um termo subscrito.

Um exemplo prático para esta tag poderia ser:

Código:

<p>Poderia usar o H<sub>2</sub>O de novo mas vou usar o CO<sub>2</sub> para variar.</p>

Resultado:

Poderia usar o H2O de novo mas vou usar o CO2 para variar.

sup
A tag sup é usada para fazer um termo sobrescrito.

Um exemplo prático para esta tag poderia ser:

Código:

<p>O número 2<sup>10</sup> é igual a 1024</p>

Resultado:

O número 210 é igual a 1024

td
A tag td é usada junto a tags th e tr em uma tabela, sua função na tabela é cria um novo campo na linha da tabela.

Um exemplo prático para esta tag poderia ser:

Código:

<table style="border: 1px solid black;">
    <thead>
        <th colspan="2" id="ex" scope="colgroup" style="border: 1px solid black;">cabeçario</th>
    </thead>
    <tbody>
        <tr>
            <td colspan="2" style="border: 1px solid black;">A</td>
        </tr>
        <tr>
            <td rowspan="2" style="border: 1px solid black;">B</td>
            <td style="border: 1px solid black;"> C</td>
        </tr>
        <tr>
            <td headers="ex" style="border: 1px solid black;">D</td>
        </tr>
    </tbody>
</table>

Resultado:

cabeçario
A
B C
D

Esta tag possui alguns atributos que podem ser utilizados como:

O atributo colspan faz a mesclagem das celulas da colunas da tabela.

O atributo rowspan faz a mesclagem das celulas da linha da tabela.

O atributo headers faz com que a celula seja relacionada a um cabeçario sem necessáriamente esteja em sua coluna ou linha.

th
A tag th é usada junto a tags thead, tr e td em uma tabela, sua função na tabela é cria um novo campo na linha do cabeçario da tabela.

Um exemplo prático para esta tag poderia ser:

Código:

<table style="border: 1px solid black;">
    <thead>
        <th colspan="2" id="ex" scope="colgroup" style="border: 1px solid black;">cabeçario</th>
    </thead>
    <tbody>
        <tr>
            <td colspan="2" style="border: 1px solid black;">A</td>
        </tr>
        <tr>
            <td rowspan="2" style="border: 1px solid black;">B</td>
            <td style="border: 1px solid black;"> C</td>
        </tr>
        <tr>
            <td headers="ex" style="border: 1px solid black;">D</td>
        </tr>
    </tbody>
</table>

Resultado:

cabeçario
A
B C
D

Esta tag possui alguns atributos que podem ser utilizados como:

O atributo colspan faz a mesclagem das celulas da colunas da tabela.

O atributo rowspan faz a mesclagem das celulas da linha da tabela.

O atributo headers faz com que a celula seja relacionada a um cabeçario sem necessáriamente esteja em sua coluna ou linha.

O atributo scope especifica se o cabeçario faz referencia apenas a coluna (col) ou linha (row) em que se encontra sua primeira celula ou se a mesclagem de todas suas colunas(colgroup) ou linhas (rowgroup) que conta.
