Comandos de Interação

Comandos de interação são comandos que criam campos a serem preenchidos ou alterados pelos usuários. Este tipo de comando é normalmente utilizado em formulários ou campos de pesquisa.

Abaixo irei citar alguns desses comandos e em que situações são usados:

a
A tag a possui a função de criar um link para outras páginas ou seções do documento.

Um exemplo prático para esta tag poderia ser:

Código:

<p>Exercício de <a href="../ProjetosHTMLCSS/CursoemVideo/exercicios/ex010/index.html" target="_blank" rel="nofollow">links</a></p>

Resultado:

Exercício de links

Esta tag possui alguns atributos que podem ser utilizados como:

O atributo download indica que, ao clicar no link, irá baixar o arquivo citado.

O atributo hrel indica o endereço do link para onde será redirecionado.

O atributo hreflang diz ao navegador em que idioma a página do link estará.

O atributo media ainda não utilizei para saber suas funcionalidades.

O atributo ping ainda não utilizei para saber suas funcionalidades.

O atributo referrerpolicy ainda não utilizei para saber suas funcionalidades.

O atributo rel diz ao navegador qual a relação da presente página com a página do link.

O atributo target define como será aberta a página do link no navegador.

O atributo type diz qual será o formato do documento presente no link.

datalist
A tag datalist cria uma lista de opções predefinidas.

Um exemplo prático para esta tag poderia ser:

Código:

<p>
    <label for="iprof">Profissão: </label>
    <input type="text" name="prof" id="iprof" list="lstprof">
    <datalist id="lstprof">
        <option>Administração</option>
        <option>Contabilista</option>
        <option>Desenvolvedor</option>
        <option>Professor</option>
    </datalist>
</p>

Resultado:

Esta tag possui alguns atributos que podem ser utilizados como:

O atributo id é o identificador da lista, usado para referenciar a um input específico.

input
A tag input cria um campo para interação do usuário que varia dependendo do atributo type.

Um exemplo prático para esta tag poderia ser:

Código:

    <p>
    <label for="iprof">Profissão: </label>
    <input type="text" name="prof" id="iprof" list="lstprof">
    <datalist id="lstprof">
        <option>Administração</option>
        <option>Contabilista</option>
        <option>Desenvolvedor</option>
        <option>Professor</option>
    </datalist>
    </p>

Resultado:

Como esta tag possui MUITOS atributos que podem ser utilizados, vou citar apenas alguns que são mais essenciais:

O atributo id é o identificador da resposta.

O atributo list indica a qual lista o campo de preenchimento deve receber a resposta.

O atributo name define o nome do campo de preenchimento.

O atributo placeholder coloca um texto de exibição no campo de preenchimento.

O atributo required faz o campo de preenchimento ser obrigatório para o envio do formulário.

O atributo type especifica o tipo de entrada que o campo irá aceitar.

label
A tag label faz o texto que se refere a um input.

Um exemplo prático para esta tag poderia ser:

Código:

    <p>
    <label for="iprof">Profissão: </label>
    <input type="text" name="prof" id="iprof" list="lstprof">
    <datalist id="lstprof">
        <option>Administração</option>
        <option>Contabilista</option>
        <option>Desenvolvedor</option>
        <option>Professor</option>
    </datalist>
    </p>

Resultado:

Esta tag possui alguns atributos que podem ser utilizados como:

O atributo for faz uma ligação entre o label e um id de um input.

option
A tag option são as opções de resposta de um select.

Um exemplo prático para esta tag poderia ser:

Código:

<p>
    <label for="iest">Estado: </label>
    <select name="estado" id="iest">
        <option value="">--- Escolha ---</option>
        <option value="MG">Minas Gerais</option>
        <option value="RJ">Rio de Janeiro</option>
        <option value="SP">São Paulo</option>
    </select>
</p>

Resultado:

Esta tag possui alguns atributos que podem ser utilizados como:

O atributo label faz a resposta mais curta aparecer na tela.

O atributo selected define que esta opção está pré-selecionada.

O atributo value define o que será enviado quando esta opção for selecionada.

select
A tag select cria uma lista de seleção.

Um exemplo prático para esta tag poderia ser:

Código:

<p>
    <label for="iest">Estado: </label>
    <select name="estado" id="iest">
        <option value="">--- Escolha ---</option>
        <option value="MG">Minas Gerais</option>
        <option value="RJ">Rio de Janeiro</option>
        <option value="SP">São Paulo</option>
    </select>
</p>

Resultado:

Esta tag possui alguns atributos que podem ser utilizados como:

O atributo id é o identificador da lista de seleção.

O atributo multiple permite selecionar múltiplas opções.

O atributo name define o nome do campo de seleção.

O atributo required faz o campo de seleção ser obrigatório para o envio do formulário.

textarea
A tag textarea cria um campo para o usuário digitar com quebra de linha.

Um exemplo prático para esta tag poderia ser:

Código:

<p>
    <label for="imsg">Mensagem: </label>
    <textarea name="msg" id="imsg" cols="30" rows="10" placeholder="Coloque sua mensagem aqui..."></textarea>
</p>

Resultado:

Esta tag possui alguns atributos que podem ser utilizados como:

Os atributos cols e rows definem o tamanho do campo de texto baseando-se em colunas e linhas.

O atributo id é o identificador do campo de texto.

O atributo name define o nome do campo de texto.

O atributo placeholder coloca um texto de exibição no campo de texto.

O atributo required faz o campo de seleção ser obrigatório para o envio do formulário.
